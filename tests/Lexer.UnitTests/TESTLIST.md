# Список тестов для лексера DEA

## 1. Ключевые слова (Keywords)
- [x] Распознавание ключевого слова `const`.
- [x] Распознавание ключевого слова `func`.
- [x] Распознавание ключевого слова `proc`.
- [x] Распознавание ключевого слова `return`.
- [x] Распознавание ключевого слова `if`.
- [x] Распознавание ключевого слова `else`.
- [x] Распознавание ключевого слова `while`.
- [x] Распознавание ключевого слова `for`.
- [x] Распознавание ключевого слова `to`.
- [x] Распознавание ключевого слова `downto`.
- [x] Распознавание ключевого слова `break`.
- [x] Распознавание ключевого слова `continue`.
- [x] Распознавание ключевого слова `true`.
- [x] Распознавание ключевого слова `false`.
- [x] Распознавание ключевого слова `input`.
- [x] Распознавание ключевого слова `print`
- [x] Распознавание ключевого слова `int`
- [x] Распознавание ключевого слова `num`
- [x] Распознавание ключевого слова `string`
- [x] Распознавание ключевого слова `bool`
- [x] Регистронезависимость ключевых слов. `fUNc INT Main`

## 2. Идентификаторы (Identifiers)
- [x] Распознавание идентификатора, начинающегося с буквы.
- [x] Распознавание идентификатора, содержащего цифры (`var1`).
- [x] Идентификатор не должен совпадать с ключевыми словами.
- [x] Регистрозависимость идентификаторов (`myVar` != `myvar`).

## 3. Литералы (Literals)
### 3.1. Числа
- [x] Целое число (`123, 45.67`).
- [x] Десятичная дробь (`1.23`).
- [x] Дробь с нулем в начале (`0.5`).

### 3.2. Строки
- [x] Однострочный строковый литерал в двойных кавычках `"text"`.
- [x] Пустая строка `""`.
- [x] Escape-последовательности:
- [x] `\"` (двойная кавычка)
- [x] `\\` (обратный слэш)
- [x] `\t` (табуляция)
- [x] `\r` (возврат каретки)
- [x] `\n` (перенос строки)
- [x] Строка с символами комментариев внутри ("#", "/* */"), чтобы проверить, что они не считаются комментарием.
`\"# not comment\" \"/* not comment */\`

## 4. Операторы (Operators)
- [x] Присваивание `=`.
- [x] Сравнение: `==`, `!=`, `<`, `<=`, `>`, `>=`.
- [x] Арифметика: `+`, `-`, `*`, `/`, `//`, `%`.
- [x] Возведение в степень `^`.
- [x] Логические: `&&`, `||`, `!`.

## 5. Прочие лексемы (Delimiters)
- [x] Разделители: `;`, `,`, `(`, `)`, `{`, `}`.

## 6. Комментарии (Comments)
- [x] Однострочный комментарий `#`.
- [x] Многострочный комментарий `/* ... */`.

## 7. Обработка ошибок
- [x] Незакрытый многострочный комментарий.
- [x] Неизвестный символ (например, `@`).
- [x] Незакрытая строка.
- [x] Некорректное целое число с ведущим нулём (`01`).
- [x] Некорректное вещественное число без дробной части (`1.`).
- [x] Дочитывание дробной части при ошибке ведущего нуля.
- [x] Одиночный символ `&` вместо `&&`.
- [x] Одиночный символ `|` вместо `||`.
- [x] Файл закончился сразу после `\`.
- [x] Неизвестный символ в escape-последовательности `\q`.

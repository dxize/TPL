# Список тестов модуля VirtualMachine (DEA)

## Остановка вычислений

- [x] Остановка виртуальной машины с нулевым кодом возврата
- [ ] Остановка виртуальной машины с ненулевым кодом возврата
- [ ] Ошибка при попытке возврата нецелочисленного значения как кода выхода

## Стековые вычисления выражений

- [x] Возврат одного значения со стека
- [x] Сложение и вычитание с помощью стека
- [x] Умножение, деление и остаток от деления (modulo)
- [x] Целочисленное деление с помощью стека
- [x] Возведение числа в степень
- [x] Унарный минус в арифметическом выражении
- [ ] Операции над вещественными числами (тип `num`)
- [ ] Конкатенация строк с помощью стека
- [ ] Удаление значения с вершины стека (`pop`)
- [ ] Ошибка при выполнении арифметики над несовместимыми типами
- [ ] Ошибка при делении на ноль

## Встроенные функции

- [ ] Функция `print` (вывод базовых типов)
- [ ] Функция `input` (чтение значения в переменную)
- [ ] Функция `len` (получение длины строки)
- [ ] Функция `substr` (извлечение подстроки)
- [ ] Функция `abs` (модуль числа)
- [ ] Функции `min` и `max` с переменным числом аргументов

## Переменные и константы

- [ ] Сохранение и загрузка значений переменных (`mutable`)
- [ ] Сохранение и загрузка значений констант (`immutable`)
- [ ] Ошибка при попытке записи значения в константу
- [ ] Ошибка при чтении из неинициализированной переменной
- [ ] Ошибка при повторном объявлении идентификатора в одной области видимости
- [ ] Ошибка при несоответствии типов при присваивании

# Список тестов модуля VirtualMachine (DEA)

- [x] Остановка виртуальной машины с нулевым кодом возврата
- [x] Остановка виртуальной машины с ненулевым кодом возврата
- [ ] Ошибка при попытке возврата нецелочисленного значения как кода выхода

- [x] Сложение и вычитание с помощью стека
- [x] Умножение, деление и остаток от деления (modulo)
- [x] Целочисленное деление с помощью стека
- [x] Возведение числа в степень
- [x] Унарный минус в арифметическом выражении
- [ ] Операции над вещественными числами (тип `num`)
- [ ] Конкатенация строк с помощью стека
- [ ] Удаление значения с вершины стека (`pop`)
- [ ] Ошибка при выполнении арифметики над несовместимыми типами
- [ ] Ошибка при делении на ноль

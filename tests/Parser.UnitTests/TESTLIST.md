# Список тестов парсера

## Выражения (expressions-grammar.md)

### Литералы

#### Числовые литералы
- [x] Целое число: `2025` => 2025
- [x] Вещественное число: `3.14` => 3.14
- [x] Ноль: `0` => 0
- [x] Дробное число: `123.456` => 123.456

#### Строковые литералы
- [x] Строка в двойных кавычках: `"hello"` => "hello"
- [x] Пустая строка: `""` => ""

### Унарные операторы (приоритет 8)

#### Унарный минус
- [x] Отрицательное целое: `-5` => -5
- [x] Отрицательное вещественное: `-3.14` => -3.14

#### Унарный плюс
- [x] Положительное целое: `+5` => 5
- [x] Положительное число: `+42` => 42

#### Логическое НЕ
- [x] Отрицание истины: `!1` => false
- [x] Отрицание лжи: `!0` => true

### Бинарные операторы

#### Возведение в степень (приоритет 7, правоассоциативный)
- [x] Простое возведение в степень: `2 ^ 3` => 8
- [x] Правая ассоциативность: `2 ^ 3 ^ 2` => 512 (= 2^(3^2) = 2^9)
- [x] С отрицательным основанием: `(-2) ^ 3` => -8
- [x] С дробной степенью: `4 ^ 0.5` => 2

#### Мультипликативные операторы (приоритет 6)
- [x] Умножение: `2 * 3` => 6
- [x] Деление: `6 / 2` => 3
- [x] Целочисленное деление: `7 // 2` => 3
- [x] Остаток от деления: `7 % 3` => 1
- [x] Деление на ноль: `5 / 0` => DivideByZeroException
- [x] Целочисленное деление на ноль: `5 // 0` => DivideByZeroException
- [x] Остаток от деления на ноль: `5 % 0` => DivideByZeroException

#### Аддитивные операторы (приоритет 5)
- [x] Сложение: `1 + 2` => 3
- [x] Вычитание: `5 - 3` => 2
- [x] Конкатенация строк: `"hello" + "world"` => "helloworld"
- [x] Конкатенация с пустой строкой: `"test" + ""` => "test"

#### Операторы сравнения (приоритет 4)
- [x] Меньше (истина): `3 < 5` => true
- [x] Меньше (равно): `5 < 5` => false
- [x] Меньше (ложь): `6 < 5` => false
- [x] Больше (ложь): `3 > 5` => false
- [x] Больше (равно): `5 > 5` => false
- [x] Больше (истина): `6 > 5` => true
- [x] Меньше или равно (меньше): `3 <= 5` => true
- [x] Меньше или равно (равно): `5 <= 5` => true
- [x] Меньше или равно (больше): `6 <= 5` => false
- [x] Больше или равно (меньше): `3 >= 5` => false
- [x] Больше или равно (равно): `5 >= 5` => true
- [x] Больше или равно (больше): `6 >= 5` => true

#### Операторы равенства/неравенства (приоритет 3)
- [x] Равенство чисел (истина): `5 == 5` => true
- [x] Равенство чисел (ложь): `5 == 3` => false
- [x] Неравенство чисел (истина): `5 != 3` => true
- [x] Неравенство чисел (ложь): `5 != 5` => false
- [x] Равенство строк (истина): `"hello" == "hello"` => true
- [x] Равенство строк (ложь): `"hello" == "world"` => false
- [x] Неравенство строк (истина): `"hello" != "world"` => true

#### Логическое И (приоритет 2)
- [x] Истина И Истина: `1 && 1` => true
- [x] Истина И Ложь: `1 && 0` => false
- [x] Ложь И Истина: `0 && 1` => false
- [x] Ложь И Ложь: `0 && 0` => false

#### Логическое ИЛИ (приоритет 1)
- [x] Истина ИЛИ Истина: `1 || 1` => true
- [x] Истина ИЛИ Ложь: `1 || 0` => true
- [x] Ложь ИЛИ Истина: `0 || 1` => true
- [x] Ложь ИЛИ Ложь: `0 || 0` => false

### Приоритет и ассоциативность операторов

#### Левая ассоциативность
- [x] Вычитание: `10 - 3 - 2` = `(10 - 3) - 2` = 5
- [x] Деление: `12 / 3 / 2` = `(12 / 3) / 2` = 2

#### Приоритет унарных операторов (8)
- [x] Унарный минус перед умножением: `-3 + 2` = `(-3) + 2` = -1

#### Приоритет возведения в степень (7)
- [x] Степень перед умножением: `2 ^ 3 * 2` = `(2 ^ 3) * 2` = 16
- [x] Умножение после степени: `2 * 3 ^ 2` = `2 * (3 ^ 2)` = 18

#### Приоритет мультипликативных операторов (6)
- [x] Умножение перед сложением: `2 + 3 * 4` = `2 + (3 * 4)` = 14
- [x] Умножение перед сложением: `3 + 2 * 3` = `3 + (2 * 3)` = 9

#### Приоритет аддитивных операторов (5)
- [x] Сложение перед сравнением: `1 + 2 > 2` = `(1 + 2) > 2` = true

#### Приоритет операторов сравнения (4)
- [x] Сравнение перед равенством: `1 < 2 == 5 > 4` = `(1 < 2) == (5 > 4)` = true

### Скобки
- [x] Скобки изменяют приоритет: `(1 + 2) * 3` => 9
- [x] Скобки изменяют приоритет: `2 * (3 + 4)` => 14
- [x] Вложенные скобки: `((1 + 2) * 3) + 4` => 13

### Встроенные функции

#### abs(x) - модуль числа
- [x] Модуль отрицательного: `abs(-5)` => 5
- [x] Модуль положительного: `abs(5)` => 5
- [x] Модуль дробного: `abs(-3.14)` => 3.14
- [ ] Модуль без аргументов: `abs()` => ArgumentException
- [ ] Модуль с двумя аргументами: `abs(1, 2)` => ArgumentException

#### min(x, y, ...) - минимум
- [x] Минимум из трех: `min(7, 3, 5)` => 3
- [x] Минимум из одного: `min(1)` => 1
- [x] Минимум из двух: `min(5, 2)` => 2
- [ ] Минимум без аргументов: `min()` => ArgumentException

#### max(x, y, ...) - максимум
- [x] Максимум из трех: `max(2, 8, 4)` => 8
- [x] Максимум из одного: `max(1)` => 1
- [x] Максимум из двух: `max(5, 2)` => 5
- [ ] Максимум без аргументов: `max()` => ArgumentException

#### Комбинированные выражения с функциями
- [x] Функция в выражении: `abs(-5) + 3` => 8
- [x] Функция в выражении: `min(10, 20) * 2` => 20
- [x] Функция со степенью: `max(1, 2, 3) ^ 2` => 9
- [x] Функция в скобках: `(1 + 2) * abs(-3)` => 9

### Обработка ошибок

#### Несовместимые типы
- [x] Сложение числа и строки: `1 + "hello"` => Exception
- [x] Вычитание строки и числа: `"world" - 5` => Exception
- [x] Умножение строки на число: `"test" * 2` => Exception
- [x] Деление строки на число: `"abc" / 3` => Exception



## Инструкции верхнего уровня (top-level-grammar.md)

### Объявление переменных

#### Объявление без инициализации
- [x] Объявление переменной: `var x;`

#### Объявление с инициализацией
- [x] Объявление с числовым литералом: `var x = 10;`
- [x] Объявление с числовым литералом: `var x = 10; print(x);`
- [x] Объявление с выражением: `var x = 2 + 3; print(x);`
- [x] Объявление с использованием переменной: `var a = 10; var b = 20; var sum = a + b; print(sum);`

### Объявление констант

#### Объявление константы
- [x] Объявление константы: `const PI = 3.14;`
- [x] Использование константы в выражении: `const PI = 3.14; var r = 2; print(PI * r * r);`

### Присваивание

#### Присваивание значения
- [x] Присваивание числового литерала: `var x; x = 20; print(x);`
- [x] Присваивание выражения: `var x; var y = 5; x = y * 2; print(x);`
- [x] Присваивание с использованием встроенных функций: `var x; x = abs(-10); print(x);`

### Ввод

#### Ввод в переменную
- [x] Ввод в объявленную переменную: `var x; input(x);`
- [x] Ввод с последующим использованием: `var x; input(x); print(x);`

### Вывод

#### Вывод значений
- [x] Вывод числа: `print(10);`
- [x] Вывод переменной: `var x = 5; print(x);`
- [x] Вывод выражения: `var x = 10; print(x + 5);`
- [x] Вывод нескольких аргументов: `print(1, 2 + 3);`
- [x] Вывод без аргументов: `print();`
- [x] Вывод с использованием встроенных функций: `print(min(10, 3, 15)); print(max(1, 5)); print(abs(-5));`

### Последовательность инструкций

#### Программа с несколькими инструкциями
- [x] Несколько переменных и операций: `var a = 10; var b = 20; var sum = a + b; print(sum);`
- [x] Комплексная программа: `const PI = 3.14; var r = 2; print(PI * r * r);`
- [x] Программа с вводом-выводом: `var x; input(x); print(x);`
- [x] Арифметические операции в инструкциях: `var x = 15; var y = 3; var a = x * y; var b = x / y; print(a); print(b);`
- [x] Выражения со скобками: `var result = (10 + 5) * 2; print(result);`
- [x] Возведение в степень в инструкции: `var base = 2; var poow = 8; var power = base ^ poow; print(power);`
- [x] Операции сравнения: `var a = 10; var b = 5; print(a > b); print(a < b); print(a == b);`
- [x] Логические операции: `var x = 1; var y = 0; print(x && y); print(x || y);`
- [x] Целочисленное деление и остаток: `var a = 17; var b = 5; var c = a // b; var d = a % b; print(c); print(d);`

### Обработка ошибок

#### Деление на ноль
- [x] Деление на ноль: `var x = 1; print(x / 0);` => DivideByZeroException
- [x] Целочисленное деление на ноль: `var x = 5; print(x // 0);` => DivideByZeroException
- [x] Остаток от деления на ноль: `var x = 5; print(x % 0);` => DivideByZeroException

#### Использование необъявленных переменных
- [x] Использование необъявленной переменной (ошибка): `print(x);` => Exception

### LW5

### Ветвление (if-else)
- [x] Простейший if: `if (x > 5) { print(1); }`
- [x] Инструкция if-else: `if (x > 5) { print(1); } else { print(0); }`
- [x] Вложенные if: `if (x > 5) { if (y > 10) { print(1); } else { print(2); } }`

### Циклы
- [x] Цикл while: `while (i < 3) { i = i + 1; print(i); }`
- [x] Цикл for to: `for (i = 1 to 3) { print(i); }`
- [x] Цикл for downto: `for (i = 3 downto 1) { print(i); }`
- [x] Инструкция break (while): `while (1) { i = i + 1; if (i == 3) { break; } print(i); }`
- [x] Инструкция continue (while): `while (i < 3) { i = i + 1; if (i == 2) { continue; } print(i); }`
- [x] Инструкция break (for): `for (i = 1 to 5) { if (i == 3) { break; } print(i); }`
- [x] Инструкция continue (for): `for (i = 1 to 3) { if (i == 2) { continue; } print(i); }`

### Функции и процедуры
- [x] Объявление и вызов функции: `func add(a, b) { return a + b; }`
- [x] Объявление и вызов процедуры: `proc log(val) { print(val); }`
- [x] Инструкция return: `return a + b;`

### Семантические ошибки
- [x] Повторное объявление переменной: `var x = 1; var x = 2;` => Exception
- [x] Константа без инициализации: `const PI;` => Exception
- [x] Присваивание константе: `const PI = 3.14; PI = 3.15;` => Exception
